Cómo usar Cada plantilla es un archivo HTML autocontenido que enlaza la hoja de estilos común (docs.css). Para exportar a PDF, abra la plantilla y use el botón Imprimir / Guardar PDF de la esquina superior derecha (o Cmd / Ctrl + P). Los datos entre llaves {{ }} indican variables a sustituir por su sistema de gestión.
