# Fonts

This system uses two Google Fonts loaded via the CDN import at the top of `colors_and_type.css`:

- **Montserrat** — display / headings (weights 500 / 600 / 700 / 800)
- **Source Sans 3** — body / UI (weights 400 / 500 / 600 / 700)

No font files are bundled in this repository — fonts load over the network from `fonts.googleapis.com`. For an offline production build, self-host them from <https://fonts.google.com/specimen/Montserrat> and <https://fonts.google.com/specimen/Source+Sans+3> (both SIL Open Font License 1.1).

## Substitution flag

The logo wordmark "ESTUDIO BACIGALUPO" appears to be set in a geometric sans of the **Montserrat / Gotham / Avenir** family. Without access to the original artwork files we cannot confirm the exact typeface. **Montserrat** is used as the documented substitute for headings throughout the system. If the firm holds a license to the specific wordmark typeface, please share the font files and we will swap them in.
